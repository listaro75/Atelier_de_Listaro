Test Upload Multiple Images - Admin

Cette page permet de tester le système d'upload multiple d'images dans l'admin.

Test d'ajout de produit avec plusieurs images

Sélectionnez plusieurs images. La première sera l'image principale.

Instructions de test

  1. Sélectionnez plusieurs images (2-5 images recommandées)
  2. Vérifiez que l'aperçu s'affiche correctement
  3. Notez que la première image a le badge "Principal"
  4. Testez la suppression d'images avec le bouton ×
  5. Cliquez sur "Simuler l'upload" pour voir le résultat

Fonctionnalités testées

  • ✓ Upload multiple d'images
  • ✓ Aperçu en temps réel
  • ✓ Identification de l'image principale
  • ✓ Suppression d'images avant upload
  • ✓ Validation des formats d'image
  • ✓ Interface utilisateur intuitive
